# HBI基础 (Hybrid Bonding Interface)

> [!def] 定义 (Definition)
> 混合键合(Hybrid Bonding)是一种将Cu-Cu金属键合与介质键合结合的先进封装技术，实现芯片间的垂直互连，无需传统焊料凸点。

## 📖 基本原理 (Fundamentals)

### 工艺背景

传统微凸点(micro-bump)互连：
- 使用焊料凸点连接上下层芯片
- Pitch受限于焊料工艺，通常≥40μm
- 存在IMC(金属间化合物)层，增加接触电阻

混合键合改进：
- **Cu-Cu直接键合**：金属铜直接接触，无焊料
- **介质键合**：SiO₂或SiCN等介质材料同时键合
- 实现更小pitch（可低至10μm以下）

### 键合结构

```
┌─────────────────┐
│    上层芯片      │
├─────────────────┤
│  Cu焊盘 │ SiO₂  │  ← 键合界面
├────────┼────────┤
│  Cu焊盘 │ SiO₂  │
├─────────────────┤
│    下层芯片/晶圆  │
└─────────────────┘
```

### 物理机制

1. **表面活化**：等离子体处理使表面具有活性
2. **室温预键合**：介质层先接触贴合
3. **热退火**：高温下Cu扩散形成金属键合

## 🔗 与HBI的关系 (Relevance to HBI)

### 在混合键合中的作用

- HBI即指混合键合形成的互连界面
- 界面质量直接影响电学性能和可靠性

### 影响因素

| 因素 | 影响 |
|------|------|
| 表面平整度 | 键合质量、空洞率 |
| 键合温度 | Cu晶粒生长、键合强度 |
| 退火时间 | 接触电阻、可靠性 |
| Cu表面氧化 | 接触电阻 |

## 📊 关键参数 (Parameters & Metrics)

| 参数 | 典型值 | 单位 | 说明 |
|------|--------|------|------|
| Pitch | 10-40 | μm | 键合点间距 |
| Cu焊盘直径 | 5-20 | μm | 通常为pitch的50-70% |
| 键合温度 | 250-350 | °C | 退火温度 |
| 接触电阻 | 50-200 | mΩ | 单点接触电阻 |
